## 💬 Basic Chat (5 Steps)

### Step 1: Select Subject
```
┌──────────────────────────────────┐
│ Subject: [▼ DBMS            ]    │
│           • General              │
│           • DBMS                 │
│           • OS                   │
│           • Computer Networks    │
│           • DSA                  │
│           • Web Dev              │
└──────────────────────────────────┘
```

Click dropdown → Choose your subject

### Step 2: Type Your Question
```
┌──────────────────────────────────────────┐
│ Type here...                             │
│ "What is database normalization?"        │
└──────────────────────────────────────────┘
```

### Step 3: Send or Speak
```
┌────────────────┬──────────┬────────────┐
│ Your question  │ [Send]   │ 🎤         │
└────────────────┴──────────┴────────────┘
```

**Option A:** Click "Send" button
**Option B:** Click 🎤 button to speak (say your question)

### Step 4: Wait for AI Response
```
You:  "What is database normalization?"
      [thinking bubble appears]

AI is thinking... ⏳ (1-3 seconds)
```

### Step 5: See the Answer
```
You:  "What is database normalization?" [blue, right]

AI:   "Database normalization is a process..." [gray, left]
      [auto-scrolls to show]
```

### Pro Tips for Voice
✓ Speak slowly and clearly
✓ Wait for "Listening..." before speaking
✓ Works best with Chrome/Edge (not Safari)
✓ Allow microphone when browser asks
✓ Speak in a quiet room

## 🎯 Common Questions

### Q: Can my friends see my chats?
**A:** No! Each user sees only their own chats. Your privacy is safe.

### Q: Does AI remember me?
**A:** Yes! Same subject = same chat history.

### Q: Can I delete a chat?
**A:** Not yet in the UI, but backend supports it. Contact admin.

### Q: Is the AI always correct?
**A:** AI is smart but not perfect. Always verify important info!

### Q: What if I ask something off-topic?
**A:** AI will try to answer, but it's trained for academics.

### Q: Can AI help with homework?
**A:** AI explains concepts. You solve problems yourself!

### Q: How fast is the AI?
**A:** Usually 1-3 seconds per response.

### Q: Does it work offline?
**A:** No, but you get a message and it saves locally.
